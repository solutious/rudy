update_dns (rset)

Add instance info to machine and save it. This is really important for the initial startup so the metadata is updated right away. But it’s also important to call here because if a routine was executed and an unexpected exception occurs before this update is executed the machine metadata won’t contain the DNS information. Calling it here ensure that the metadata is always up-to-date. Each Rye:Box instance has a Rudy::Machine instance in its stash so rbox.stash.refresh! == machine.refresh!
